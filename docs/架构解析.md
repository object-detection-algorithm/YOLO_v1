
# 架构解析

使用`Python`实现，相关源码位于`py`目录下

```
├── batch_detect.py
├── data
├── detector.py
├── lib
    ├── data
    │   ├── __init__.py
    │   └── parse_location.py
    ├── __init__.py
    ├── models
    │   ├── basic_conv2d.py
    │   ├── __init__.py
    │   ├── location_dataset.py
    │   ├── multi_part_loss.py
    │   ├── __pycache__
    │   └── yolo_v1.py
    ├── __pycache__
    ├── train.py
    └── utils
        ├── draw.py
        ├── file.py
        ├── __init__.py
        ├── __pycache__
        ├── util.py
        └── voc_map.py
└── models
```

* 数据集操作：`lib/data/parse_location.py`
* 模型定义：`lib/models`
* 训练文件：`lib/train.py`
* 批量测试：`batch_detect.py`
